## Installation, Why?

_Absolutely we need to install TypeScript because it also installs the TypeScript Compiler, which compiles TypeScript code to JavaScript._

## Steps for installation 👇👇

1. Click [here](https://www.typescriptlang.org) to go to the typescript website.
2. Click on Download and scroll to the globally installing TypeScript
3. Make sure you have already installed Node JS. Then run the command `npm install -g typescript` in your local maching using command prompt.
4. To check whether it's been installed or not you can run the command `tsc --version` If you are getting something like this `Version 6.0.3` then it is installed.
